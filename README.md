# Inclind Storybook

Storybook project to be packaged and reused. This Storybook is setup to use web components specifically.

##### INSTALLATION

`npm install`

`npm run storybook`

##### TROUBLESHOOTING

If you're getting errors such as `globalThis is not defined`, try the following:

`nvm install 12.17`

`nvm use 12.17`
